HTML SEMANTIC TAGS

Tag Name Description
Element placement
<header> Header Describes the content at the top of the page
<nav> Navigation Encapsulates the page’s navigational links.
<main> Main Encapsulates the main content of a page
<footer> Footer Includes the page’s footer content at the bottom
Embedding media
<video> Video Allows us to add videos to our website.
<audio> Audio Allows us to implement audio into our website.
<embed> Embed Can be used to implement any type of media.
Figure and Figcaption
<figure> Figure Is used to encapsulate media such as an image, diagram. or code snippet.
<figcaption> Figcaption Is used to describe the media encapsulated within the figure element.
Section and article
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, and comments.
Aside Element
<aside> Aside Is used to mark additional information that can enhance another element but isn’t required in order to understand the main content
Dates and times
<time> Time Represents either a time of day or a calendar date.
Address
<address> Address It defines contact information for the author of the article or web page in question.
